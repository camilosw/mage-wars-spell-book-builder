# Mage Wars spellbook builder made in React

This is a clone of [www.spellbookbuilder.com](http://www.spellbookbuilder.com) made by [grga.curkovic](http://forum.arcanewonders.com/index.php?topic=17116.0).

## Motivation

The original spellbook builder is not mobile friendly and the spellbooks are saved on the device local storage, so you can't edit it on your computer and retrieve it on your mobile device.
